# User Stories

## Customer

As a customer,
I want to scan a QR code,
so that I can see the menu.

---

As a customer,
I want to add items to my cart,
so that I can place an order.

---

As a customer,
I want to track my order,
so that I know when it is ready.

---

## Chef

As a chef,
I want to receive new orders instantly,
so that I can start preparing them.

---

As a chef,
I want to change order status,
so that customers know the progress.

---

## Admin

As an admin,
I want to manage menu items,
so that the menu stays updated.

---

As an admin,
I want to manage tables,
so that QR codes can be generated.

---

As an admin,
I want to see reports,
so that I can monitor the restaurant.
